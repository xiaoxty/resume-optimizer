# Resume Rewrite Examples

Real before/after examples for common roles. Use these as reference when rewriting.

---

## Example 1: Software Engineer (3 years exp, targeting US market)

### Before

```
Work Experience

Software Engineer — ABC Tech, 2022–2025
- Worked on backend systems using Java
- Helped with code reviews for the team
- Was responsible for database work
- Participated in agile meetings
- Fixed bugs and did testing
```

### After

```
Work Experience

Software Engineer — ABC Tech, 2022–2025
- Architected and shipped 4 high-throughput REST APIs (Java/Spring Boot) 
  serving 80K+ daily requests with 99.9% uptime
- Conducted 150+ peer code reviews, driving a 40% reduction in post-release 
  defects over 18 months
- Optimized 15 slow PostgreSQL queries via indexing and query rewriting, 
  cutting average API latency from 420ms to 85ms (-80%)
- Championed shift to trunk-based development; reduced deployment cycle 
  from 2 weeks to daily releases
```

**Changes made:**
- Added tech stack (Java/Spring Boot, PostgreSQL)
- Quantified every bullet (80K requests, 150 reviews, 40% defect reduction, etc.)
- Replaced passive phrases ("was responsible for") with strong action verbs
- Removed "participated in agile meetings" — too generic, adds no value

## Example 2: Marketing Manager (5 years exp, targeting Singapore)

### Before

```
Marketing Manager — XYZ Brand, 2020–2025
- Managed social media accounts for the company
- Ran advertising campaigns on various platforms
- Worked with the product team on launches
- Created content for the brand
- Analyzed marketing data
```

### After

```
Marketing Manager — XYZ Brand, 2020–2025
- Grew brand's Instagram following from 18K to 112K (+522%) in 24 months 
  through data-driven content strategy and influencer partnerships
- Managed SGD 480K annual digital ad budget across Meta, Google, and TikTok; 
  achieved 4.7× ROAS, exceeding KPI by 35%
- Led go-to-market strategy for 4 product launches, generating SGD 2.1M in 
  pipeline within 90 days of each launch
- Produced 200+ content assets (video, copy, design briefs) per quarter; 
  top-performing video reached 3.2M organic views
- Built weekly marketing analytics dashboard (Looker Studio) reducing 
  reporting time by 6 hours/week
```

**Changes made:**
- Every bullet now has a number or result
- Currency localized to SGD (Singapore target market)
- "Various platforms" → specific platforms named (better for ATS)
- Weak verbs ("worked with", "created") → strong verbs ("led", "built", "produced")

## Example 3: Fresh Graduate / No Work Experience (targeting UK)

### Before

```
Education
BSc Computer Science, University of XYZ, 2021–2024

Experience
- Did a 3-month internship at a tech company
- Completed final year project on machine learning
- Member of the coding club
```

### After

```
Education
BSc Computer Science, University of XYZ, 2021–2024  
First Class Honours | Relevant modules: Machine Learning, Distributed Systems, Algorithms

Projects & Experience

Software Engineering Intern — ByteStart Ltd, Jun–Aug 2023
- Contributed 2 production features (user notification system + API rate limiter) 
  to a Node.js/MongoDB codebase serving 50K daily active users
- Reduced CI pipeline build time by 30% by parallelising test suites

Final Year Project: Sentiment Analysis Engine (2024)
- Built BERT-based NLP model achieving 91.3% accuracy on custom dataset of 
  50K product reviews; outperformed baseline (TF-IDF) by 18 percentage points
- Open-sourced on GitHub: 140+ stars, 20+ forks within 3 months of publication

Coding Society — Lead Organiser, 2022–2024
- Organised 12 workshops and 2 hackathons (180+ participants); secured 
  £2,000 in sponsorship from 3 tech companies
```

**Changes made:**
- Added degree classification and relevant modules
- Expanded internship from 1 vague line to 2 strong bullets with context
- Final year project now has metrics (accuracy, GitHub stars)
- Club membership → concrete leadership with numbers

## Example 4: Career Changer (Finance → Product Management)

### Before

```
Financial Analyst — Big Bank, 2019–2024
- Analyzed financial data and created reports
- Worked with different departments
- Used Excel and PowerPoint for presentations
```

### After (repositioned for PM roles)

```
Financial Analyst → Associate Product Manager — Big Bank, 2019–2024
- Partnered with engineering and risk teams to define requirements for an 
  internal portfolio reporting tool; delivered on-time to 200+ analyst users
- Synthesized data from 5 business units into executive dashboards 
  (Excel/Tableau); reduced weekly reporting cycle from 3 days to 4 hours
- Identified UX friction in legacy trade reconciliation workflow; proposed 
  and validated 3 improvements that reduced manual entry errors by 62%
- Facilitated 30+ cross-functional alignment sessions between 
  Finance, Tech, and Compliance stakeholders
```

**Strategy for career changers:**
- Reframe existing experience using PM language (requirements, roadmap, stakeholders, users)
- Surface any product-adjacent work: tools built, processes improved, user feedback collected
- Add a strong summary at the top explicitly stating the career transition and transferable skills
