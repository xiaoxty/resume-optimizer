# Industry-Specific Resume Templates

Use these templates as starting structures. Fill in content then apply rewriting rules from SKILL.md.

---

## Quantification Cheat Sheet

When users say "I don't have any numbers", use these prompts:

| Question to ask | Possible metric |
|---|---|
| "How big was your team?" | Team size |
| "How many users/customers did it affect?" | User/customer count |
| "How much time did it save?" | Hours/week saved |
| "What was the budget?" | Dollar/currency amount |
| "How did it compare to before?" | % improvement |
| "Did you hit your targets?" | % of goal achieved |
| "How quickly did you ship it?" | Timeline vs plan |
| "Was your result better than average?" | Ranking / percentile |
